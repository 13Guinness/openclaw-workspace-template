## Quick Start

Read [[index]] — explains how shortcodes work in Breakdance.

## When to Use

✅ **USE when:**
- Using shortcodes in Breakdance
- Dynamic content (year, phone, email)
- ACF fields in Breakdance
- User-specific content

❌ **DON'T use when:**
- Not using Breakdance
- No shortcodes needed
- Complex HTML shortcodes (may break)

## Key Feature

**Zero configuration:**
1. Activate plugin
2. Shortcodes work everywhere
3. That's it

## Examples

- `[year]` in headings → auto-updating copyright
- `[phone]` in buttons → centralized contact info
- `[acf field="address"]` → ACF integration
