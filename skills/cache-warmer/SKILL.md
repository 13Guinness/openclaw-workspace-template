## When to Use

✅ **USE when:**
- Cache warming after deployments
- Scheduled cache maintenance
- Post-publish cache warming
- Performance optimization

❌ **DON'T use when:**
- No page cache configured
- Static site (no cache layer)

## Key Features

- Automatic sitemap crawling
- Scheduled warming (cron)
- Post-deploy triggers
- Rate limiting
- Progress tracking

## Hosting Compatibility

Works with:
- WP Engine, Kinsta
- Any page cache plugin
- Server-level caching
- Cloudflare APO
