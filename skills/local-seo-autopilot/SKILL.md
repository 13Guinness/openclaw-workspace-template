## When to Use

✅ **USE when:**
- Running local SEO audits
- Analyzing competitors
- Generating schema markup
- Creating GBP posts
- Finding content gaps
- Researching local keywords
- Working on plugin code

❌ **DON'T use when:**
- National/global SEO (not local)
- Non-WordPress sites
- Manual SEO preferred
