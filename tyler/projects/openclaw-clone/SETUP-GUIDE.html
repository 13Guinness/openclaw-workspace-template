3. Step-by-Step Setup

Step 1: Install Prerequisites

Install Homebrew (Package Manager)

  1. Open Terminal (Applications → Utilities)
  2. Paste this command and press Return:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  1. Follow the prompts (press Return when asked)
  2. Add Homebrew to your PATH:
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
  1. Wait for installation to complete (5-10 minutes)

Install Node.js and GitHub CLI

brew install node gh

Step 2: Authenticate with GitHub

gh auth login
  • Choose GitHub.com → HTTPS → Yes (authenticate git)
  • Choose Login with a web browser
  • Copy the one-time code from Terminal, paste it in the browser
  • Complete login (use your normal GitHub sign-in method)

Step 3: Get Your API Keys

Anthropic API Key

  1. Go to console.anthropic.com
  2. Sign up or log in
  3. Click “Get API keys” or “Create new key”
  4. Copy the key (starts with sk-ant-)
  5. Save it — you’ll paste it into the script

OpenRouter API Key

  1. Go to openrouter.ai/keys
  2. Sign up or log in
  3. Click “Create new key”
  4. Copy the key (starts with sk-or-v1-)
  5. Save it

OpenAI API Key

  1. Go to platform.openai.com
  2. Sign up or log in
  3. Go to API keys section
  4. Click “Create new secret key”
  5. Copy the key (starts with sk-)
  6. Save it immediately — OpenAI only shows it once!

Telegram Bot Token

  1. Open Telegram on your phone or computer
  2. Search for @BotFather
  3. Start a chat and type: /newbot
  4. Follow prompts:
    • Name your bot (e.g., “Tyler”)
    • Choose a username ending in “bot” (e.g., tyler_assistant_bot)
  5. BotFather will give you a token (long string of letters and numbers)
  6. Save it — this is how your agent receives messages

Step 4: Clone the Workspace

git clone https://github.com/13Guinness/openclaw-workspace-template ~/.openclaw/workspace

Step 5: Run Setup

cd ~/.openclaw/workspace/elon/projects/openclaw-clone
./setup.sh

Enter your API keys when prompted: - Paste each key and press Return - For optional items, just press Return to skip

Wait for setup to complete (5-10 minutes).

Save the credentials shown at the end! - Gateway token - Gateway password - Dashboard URL

Step 6: Run Post-Setup

./post-setup.sh

This sets up: - Automated jobs (daily briefing, email cleanup, urgent alerts) - Memory monitoring - Screenshot sync - Optional life-audit-app (if you want it locally)

Step 7: Configure Telegram

  1. Open Telegram
  2. Search for your bot (the username you created, e.g., @tyler_assistant_bot)
  3. Send: /start
  4. Your agent should respond! Try: “Hello, what time is it?”

Step 8: Customize Your Agent

Edit these files in TextEdit (or any text editor):

  1. IDENTITY.md — Agent identity is already set to “Tyler” (leave as-is)
  2. USER.md — Fill in YOUR details (name: Tyler, your preferences, timezone)
  3. SOUL.md — Agent personality is already Tyler Durden (leave as-is unless you want tweaks)

Files are located at: ~/.openclaw/workspace/

5. Troubleshooting

“command not found: brew”

Homebrew didn’t install correctly. Try:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"

“Permission denied” when running setup.sh

Make the script executable:

chmod +x setup.sh
./setup.sh

Telegram bot not responding

  1. Check that you copied the bot token correctly
  2. Verify the token in ~/.openclaw/openclaw.json
  3. Try sending /start again
  4. Check that gateway is running: openclaw gateway status

Gateway won’t start

Check logs:

cat ~/.openclaw/logs/gateway.err.log

Common issues: - Missing API key - Port 18789 already in use - Node.js not installed

Need to restart gateway

launchctl unload ~/Library/LaunchAgents/ai.openclaw.gateway.plist
launchctl load ~/Library/LaunchAgents/ai.openclaw.gateway.plist
